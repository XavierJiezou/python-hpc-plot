## Install

```bash
pip install matplotlib
```

## Usage

```bash
git clone https://github.com/XavierJiezou/python-hpc-plot.git
cd python-hpc-plot
```

Overwrite the dir `./data/data*` and run `python ./examples/main*.py` to get your figure like demo fig *.
